# Trans Security - Website Perusahaan Jasa Keamanan

## Vision
Website multi-halaman untuk PT Trans Kontinental Indonesia (Trans Security) - perusahaan konsultan keamanan dan penyedia tenaga security profesional berizin Mabes Polri. Target audiens: korporat, pabrik, mall, perbankan, dan perumahan yang membutuhkan solusi keamanan terintegrasi dan terpercaya.

## Design
**Color System:**
- `--primary: 210 80% 15%` (Navy #0A2342 - dominan untuk navbar, footer, section headers)
- `--accent: 45 69% 47%` (Gold #C9A227 - CTA buttons, highlights, borders on hover)
- `--background: 0 0% 100%` (White #FFFFFF)
- `--muted: 210 29% 97%` (Grey #F5F7FA - alternating section backgrounds)
- `--foreground: 222 47% 11%` (Dark text)

**Typography:**
- Headings: Montserrat (weights: 700 Bold)
- Body: Inter (weights: 400 Regular, 500 Medium, 600 SemiBold)

**Style Direction:** Modern corporate-premium dengan tactical precision - navy-gold palette menciptakan kesan profesional enterprise level. Animasi purposeful (Ken Burns slides, parallax sections, counter animations, smooth hover effects). Mobile-first responsive dengan hamburger slide-in menu.

## Features
1. **Home:** Hero Ken Burns 3-slide banner, animated counters, layanan unggulan cards, parallax "Mengapa Kami" & CTA, logo carousel klien
2. **About:** Profil perusahaan, timeline animasi, legalitas, tim direktur
3. **Services:** 6 layanan grid dengan hover scale, detail pages dengan form penawaran
4. **Clients:** Logo gallery grayscale-to-color, testimoni slider, studi kasus
5. **Career:** Job listings accordion, alur rekrutmen, form lamaran
6. **Contact:** Form + Google Maps embed, info kontak lengkap
7. **Global:** Sticky navbar blur on scroll, smooth scroll, WhatsApp floating button
